Fugu

Procedural geometry with Lua.

Download

Fugu runs on Windows and OS X. It is alpha software and will surely contain bugs, but it should work well enough for you to enjoy it, so check it out and please report any problems you have.

Windows (v0.2.2) OS X (v0.2.1)

Info

To learn Fugu, you can access a number of examples from the File/Examples menu in the software. There is also an online function reference and a discussion site.
