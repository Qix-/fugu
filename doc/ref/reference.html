fugu reference

transform

R 
R(radians:number,axis:vec3) generates a matrix that will rotate a point a number of radians around the specified axis
Rv 
Rv(a:vec3,b:vec3) generates a matrix that will rotate vector a to align with vector b
S 
S(s:vec3) generates a matrix that scales a point by the vector s
T 
T(t:vec3) generates a matrix that translates a point by the vector t
scalef 
scalef(poslist,center,amount): scales a list of faces around a specified point
	amount 	= 0 -> all vertices will be moved to the center point
   			= 1 -> all vertices will remain in place
   			in (0,1) -> faces will shrink
   			>1 -> faces will grow
transformf 
transformf(poslist,T) applies transform T to each vertex in poslist

		mat4
		

			
mat4 represents a 4d matrix. Example usage:
	creation: m = mat4()
	set to identity: m:set_basis(vec3(1,0,0),vec3(0,1,0),vec3(0,0,1))
	element access: m:get(row,col) row=0..3,col=0..3
	operators: m*2, m/3, m+n, m*n, m-n, -m
	make transformation matrix: 
		m:set_rotate_rad(angle,x,y,z)
		m:set_rotate_rad(angle,axis)
		m:set_translate(x,y,z)
		also see global functions R,Rv,T,S for shorthand

	Extrudes all neighbouring faces in the direction given (or the normal if dir is omitted), and re-orients the

	Note: Assumes direction is normalised.
