Detailed Description

A VertexProxyList maintains a list of vertexProxies and is responsible for updating the internal pointers when the vertices change location or notifying proxies if vertices are deleted..

XXX: Need to accommodate VCG's Allocator PointerUpdater design. i.e., be compatible with a std::vector<VertexPointer*>...
